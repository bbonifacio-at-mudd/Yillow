References

References

It's important to acknowledge that every project stands on the shoulders of giants, and Yillow is no different. Yillow made use of numerous companies, libraries, and tools for which the project would have been impossible without. This page pays tribute to all the sources that helped shape this project and serves as a "Thank you" to the amazing open-source attitudes of organizations here.

Zillow

Of course, the biggest acknowledgement goes to Zillow, the primary resource for this project. Their exhaustive housing data forms the backbone of the analyses and predictions in 'Yillow'. Their website served as an extensive reference throughout the project, and the Zestimate tool was an invaluable metric with which to compare the Yestimate to. Of course, the Zillow website gave me a clear theme to style Yillow in, and the navigation bar you see is a (heavily) edited version of the navigation bar on Zillow's own website. Zillow is an amazing company with incredible potential to make the real-estate market more transparent. Thank you Zillow!

OpenAI & AutoGPT

Expressing gratitude towards OpenAI and its GPT models, GPT-3.5 and GPT-4, is essential. These AI models have been incredibly helpful throughout the project, especially while dealing with the challenges of frontend coding. Since data science, machine learning, and AI are my forte, these models proved invaluable in learning the intricacies of frontend coding and debugging for this website. After all of this frontend coding, I feel much more confident in my abilities to code in HTML, CSS, and JavaScript. Thank you OpenAI and AutoGPT!

Github

Immense gratitude to Github for providing a platform to host my website for free using Github Pages. This allowed Yillow to be easily accessible and interactive for all users. To learn more about the free hosting services provided by Github, you can visit Github Pages. Furthermore, Github Copilot saved me tons of time with copy-pasting and general debugging! Thank you Github!

Kaggle

A big thank you to Kaggle for their house price prediction competitions. These challenges provided me with opportunities to debug and experiment with various techniques, which was instrumental to the success of this project.
